# Odin Recipes

This project is a basic recipe website developed as part of The Odin Project curriculum.

The main objective is to practice fundamental HTML concepts by building a multi-page website with links between pages and well-structured content.

## Technologies Used
- HTML5

## Project Structure
- Main index page with links to recipes
- Individual recipe pages
- Organized folders for better structure
- Use of images, lists, and headings

## Project Features
- Navigation between pages using relative links
- Recipe pages with description, ingredients, and steps
- Proper use of semantic HTML elements
- Clean and readable HTML structure

## What I Learned
- Creating multi-page websites using HTML
- Working with relative paths and folders
- Structuring content using lists and headings
- Organizing files in a simple project structure
